# 操作符重载与临时对象

[[TOC]]

- 在C++中，操作符就是函数

## 操作符重载--操作符为成员函数

<img src="./pic/image-20221227224816992.png" alt="image-20221227224816992" style="zoom:50%;" />

- 所有成员函数隐式的带着this这个参数
- this参数指向调用这个函数的对象
- 如果是二目运算符，编译器会寻找操作符左边的对象的成员函数，并调用左边对象的成员函数，作用于左边的对象
